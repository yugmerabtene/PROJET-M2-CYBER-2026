## Niveaux de fiabilité utilisés

| Tier | Type de source | Usage recommandé |
|---|---|---|
| A | Institution, régulateur, organisme public, standard reconnu | Base de diagnostic et hypothèses fortes |
| B | Rapport sectoriel reconnu ou données opérationnelles publiées | Contexte, tendances, justification produit |
| C | Pricing public, étude cabinet, source fournisseur | Benchmark, ordre de grandeur, hypothèse à surveiller |

Une source de pricing fournisseur est utile pour comparer les modèles économiques, mais elle doit être moins pondérée qu'une source institutionnelle lorsqu'il s'agit d'établir la réalité du besoin.
