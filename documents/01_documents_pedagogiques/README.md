- [Kick-off](kickoff/KICKOFF.md) : document Word original du kick-off, transcription Markdown complète et visuel associé.

Ces documents servent de base de cadrage et de point de départ pour l'[étude de marché](../02_etude_de_marche/rendu_principal.md), le [business model](../03_business_model/rendu_principal.md) et le [business plan](../04_business_plan/rendu_principal.md).
