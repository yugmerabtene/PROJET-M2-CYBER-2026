# Architecture

Ce dossier regroupe l'architecture retenue du produit DevinciWatch.

- `rendu_principal.md` : document d'architecture de référence.
- `assets/` : schémas visuels de l'architecture logicielle, du réseau Docker et du diagramme de séquence.

Cette architecture vient après la feuille de cadrage et sert de base de conception pour le développement du produit.
