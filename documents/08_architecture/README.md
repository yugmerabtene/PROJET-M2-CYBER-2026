# Architecture

Ce dossier regroupe l'architecture retenue du produit DevinciWatch. Il décrit les composants, les flux, la topologie Docker de démonstration et les décisions techniques qui guident le développement.

- [Rendu principal](rendu_principal.md) : document d'architecture de référence.
- [Assets](assets/) : schémas visuels de l'architecture logicielle, du réseau Docker et du diagramme de séquence.

Cette architecture vient après la [feuille de cadrage](../05_feuille_de_cadrage/rendu_principal.md), la [gestion de projet](../06_gestion_de_projet/rendu_principal.md) et le [cahier des charges](../07_cahier_des_charges/rendu_principal.md). Elle sert de base de conception pour le développement du produit.
