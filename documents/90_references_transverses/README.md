# Références transverses

Ce dossier centralise les sources transverses réutilisées dans plusieurs livrables. Il garantit la traçabilité des chiffres, des constats de marché et des hypothèses mobilisées dans les documents stratégiques.

- [Registre des sources](source_register.csv) : registre complet des sources exploitées.
- [Matrice de qualité](source_quality_matrix.csv) : qualification et fiabilité des sources.

Ces fichiers servent de base de traçabilité commune à l'[étude de marché](../02_etude_de_marche/rendu_principal.md), au [business model](../03_business_model/rendu_principal.md) et au [business plan](../04_business_plan/rendu_principal.md). Ils doivent être consultés lorsqu'un chiffre ou une source externe doit être vérifié.
