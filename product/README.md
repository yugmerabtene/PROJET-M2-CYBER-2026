# DevinciWatch

Depot de rendu propre de DevinciWatch, reconstruit a partir du kick-off initial, des etudes de marche consolidees et des modeles economiques derives. Tout le contenu projet est centralise dans ce dossier `product/`.

## Site officiel

- `https://devinciwatch.com`

## Structure du depot

- `rendus/`: livrables principaux remis au jury.
- `references/`: documents pedagogiques et etudes detaillees conserves pour justification.
- `annexes/`: donnees chiffrees (CSV) conservees comme preuves et references.

## Intention

Cette branche `main` est volontairement orientee rendu. Elle conserve uniquement une structure lisible pour la remise finale, sans code produit a ce stade.

## Navigation rapide

- Etude de marche: `rendus/01_etude_de_marche.md`
- Business model: `rendus/02_business_model.md`
- Business plan: `rendus/03_business_plan.md`
- Documents pedagogiques: `references/documents_pedagogiques/kickoff/KICKOFF.md`
- Etudes detaillees: `references/etude_de_marche/README.md`
- Annexes donnees: `annexes/README.md`
